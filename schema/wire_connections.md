# 📍 Финальная и бесконфликтная карта портов BlackPill

Для двух бортов танка жестко закреплены четыре независимых аппаратных таймера:
* **TIM3** — Генерация ШИМ для Борта 1
* **TIM1** — Генерация ШИМ для Борта 2
* **TIM5** — Аппаратный счетчик энкодера Борта 1
* **TIM4** — Аппаратный счетчик энкодера Борта 2

---

## 🛠️ БОРТ 1 (Левый)

| Сигнал | Назначение | Пин | Режим (Таймер / Канал) |
| :--- | :--- | :--- | :--- |
| **RPWM_1** | ШИМ Вперед | `PA6` | TIM3, Канал 1 |
| **LPWM_1** | ШИМ Назад | `PA7` | TIM3, Канал 2 |
| **EN_1** | Включение драйвера 1 | `PA2` | Обычный GPIO |
| **Энкодер 1, Фаза А** | Провод 3 шлейфа | `PA0` | TIM5, Канал 1 |
| **Энкодер 1, Фаза B** | Провод 4 шлейфа | `PA1` | TIM5, Канал 2 |

---

## 🛠️ БОРТ 2 (Правый)

| Сигнал | Назначение | Пин | Режим (Таймер / Канал) | Примечание |
| :--- | :--- | :--- | :--- | :--- |
| **RPWM_2** | ШИМ Вперед | `PA8` | TIM1, Канал 1 | |
| **LPWM_2** | ШИМ Назад | `PA9` | TIM1, Канал 2 | |
| **EN_2** | Включение драйвера 2 | `PA3` | Обычный GPIO | |
| **Энкодер 2, Фаза А** | Провод 3 шлейфа | `PB6` | TIM4, Канал 1 | Чистый пин без конфликтов! |
| **Энкодер 2, Фаза B** | Провод 4 шлейфа | `PB7` | TIM4, Канал 2 | Чистый пин без конфликтов! |

---

## 🔌 Линии питания (Общие для всей логики)

* **Шина 5V BlackPill:** Все контакты VCC энкодеров (Провод 2 шлейфов) и VCC логики драйверов.
* **Шина GND BlackPill:** Все контакты GND энкодеров (Провод 5 шлейфов) и GND логики драйверов.
